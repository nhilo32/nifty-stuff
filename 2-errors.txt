Log in to the AWS Management Console.
Navigate to the CloudTrail service.
Go to the Event history.
Filter events by the time range around when the EC2 instance was created.
Look for the RunInstances event. This event is logged when an EC2 instance is created.
Check the userIdentity element in the event. This part of the log will contain the information about the user who made the API call, such as their IAM user or role name.
